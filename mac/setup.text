step 1:check git installation
open the termial and type following command 
git --version
 and if it displays git version in terminal git is already installed 
otherwise
brew install git 

step2:configure git 
in terminal
 set your username:
git config --global user.name"yourname"
set your email:
git config --global user.email"youremail@gmail.com"
confirm your configuration:
git config --list
